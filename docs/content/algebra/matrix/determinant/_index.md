# 行列式
## 問題設定
可換環 $R$ 上の $n$ 次正方行列 $A = (A\_{i,j})$ の **行列式** (determinant) $\\det A$ は
$$
    \\det A := \\sum_{\\sigma \\in \\mathfrak{S}\_n} \\sgn(\\sigma) A\_{1,\\sigma(1)} \\cdots A\_{n,\\sigma(n)}
$$
と定義される．ここで $\\mathfrak{S}\_n$ は $n$ 次対称群である．
