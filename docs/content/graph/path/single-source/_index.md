# 単一始点最短路
## 問題設定
有向グラフ $G=(V,E)$ および始点 $s \\in V$ が与えられる．各 $v \\in V$ について，$G$ の $s$&ndash;$v$ 最短[路](..)を求めたい．

## アルゴリズム

| 枝長 | アルゴリズム | 計算量 | 備考 |
| --- | ---------- | ----- | --- |
| 単位長 | [幅優先探索](bfs) | $\\order(\|V\|+\|E\|)$ |  |
